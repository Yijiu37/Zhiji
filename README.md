# 产品文档——知记
## 一、产品说明
知记是一款基于 Vue.js 和 Flask 的智能笔记编辑器，主要面向课堂笔记整理归纳及会议记录的用户。
通过此编辑器，用户可以使用丰富的基础编辑工具创建和编辑笔记，还可以利用AI功能提升笔记质量，例如润色、翻译、摘要、图片转文字、语音转文字、全文校正等。
此外，产品还支持实时共享协作和多种格式的导入导出功能，设置了多种模板可以直接使用，并且使用文件夹对笔记进行归纳整理，满足不同用户的需求。
